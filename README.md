# NtRaiseHardError (ntdll)
 
This is an undocumented function in ntdll.dll. Causes a BSOD (Blue Screen of Death) with the error specified by the ErrorStatus parameter.

# Build

Build with MinGW64 (from MSYS2), C++ 11.
